# The Spillover Effects of Refugee Shelters

# Overview
This project analyzes the **spillover effects** of refugee shelters (Afd) on local areas. The study examines how the presence and characteristics of refugee shelters influence various socio-economic indicators in surrounding regions.

## Datasets

### 1. `clean_afd_spillover_no_hosts.xlsx`
- **Initial cleaned dataset**
- Original data with missing values (NaNs)
- Used as the starting point for further processing

### 2. `final_afd_model_ready.xlsx`
- **Main dataset** used for analysis and modeling
- Fully processed and prepared version
- Data with missing values already removed
- Ready for statistical analysis and machine learning models

## Repository Contents
- `clean_afd_spillover_no_hosts.xlsx` — initial cleaned data
- `final_afd_model_ready.xlsx` — final dataset for modeling
- `Черкшова_андан_курсовая.ipynb` — Jupyter notebook with the full analysis
